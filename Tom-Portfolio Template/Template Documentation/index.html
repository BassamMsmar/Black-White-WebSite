A) HTML Structure - top

This template is very easy to use, it has comments all over its files and clearly visible and understandable element structure. Theme contains eight sections. The section structure is the same throughout the template. Here is the section structure.

  1. Header
  2. About
  3. Services
  4. Portfolio
  5. Clients
  6. Contact
  7. Footer

B) CSS Files and Structure - top

  1. Component.css
  2. Animated Headline.css
  3. Fontawesome.css
  4. Bootstrap.css
  5. Ionicons.css
  6. Normalize.css

The main file contains all of the specific stylings for the page. The file is separated into sections using:

	/* === Header Section === */

	some code

	/* === Main Section === */
	
	some code
	
	/* === Sidebar Section === */
	
	some code
	
	/* === Footer === */
	
	some code
	
	etc, etc.

  2. boxlayout.js

D) Add Portfolio Items - top

If you would like to change the portfolio items, you would do the following:

  1. Add your portfolio pictures to the "images" folder
  2. Put your image path in portfolio section
  3. Save changes

NOTE: Make sure that your portfolio items has the same width and height

E) Setup Single project page - top

If you would like to change the portfolio items, you would do the following:

  1. Add your portfolio pictures to the "images" folder
  2. Go to single project section
  3. Put your image path
  4. Do same on other items
  5. Save changes

NOTE: Make sure that your portfolio items has the same width and height

F) Change Image on Homepage - top

Add main image, you would do the following:

  1. Go to css folder then open file called component.css
  2. Put your image path in about section.
  3. Save changes

G) Change Header Image - top

Add Header image, you would do the following:

  1. Go to css folder then open file called component.css
  2. Put your image path in header section.
  3. Save changes

H) Add Contact Information - top

  1. Go to index.html file
  2. Find Contact Section and change information
  3. Save changes

Contact form works only in hosting environment. If you want to use it, open "mail.php" file and change recepient email address
